# Email Phishing Detection System

## Project Description
This project detects phishing emails using Machine Learning techniques.
It helps identify malicious emails that attempt to steal sensitive user data.

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Naive Bayes
- TF-IDF Vectorizer

## Project Structure
src/ - Source code files  
dataset/ - Dataset used for training/testing  
research_paper/ - Research paper documentation  
presentation/ - Project presentation  

## How to Run
1. Install requirements:
   pip install -r src/requirements.txt

2. Run the project:
   python src/phishing_detection.py

## Output
The system classifies emails as:
- Phishing
- Legitimate
